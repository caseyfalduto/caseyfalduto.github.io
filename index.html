    For the very first assignment I got to choose my favorite 5 places to vacation and created a very basic website of these locations. I got to learn the very basics of HTML coding and applied them.

    Assignment 2

    We continued to use the basic rules of HTML but we added more CSS rules to the heading to create a color change. Compared to the vacation website we changed the colors to the links to make it easier to use.

    Assignment 3

    In this assignment we learned how to insert different kinds of lists, for example, ordered, unordered, and nested. We also learned how to insert a picture and more CSS rules.

    Assignment 5-1

    After being introduced divs into our skill set, we can now customize how each section of the website can look. The colors were changed for the assignment to create a pattern.

    Assignment 5-2

    We create a more structured layout by positioning divs inside each other. We learn to change the sizes of different divs and to create a float so the divs can anchor to a specific side of the page.

    Assignment 5-3

    We divided up the website into two functioning pages and allowed two different columns on each page of information. The main focus was to have the information in the middle of the page for the reader to have an easier time using the website and make it look symmetrical.

    Assignment 6

    We had to recreate the layout of the old school IMDB page without any help from our teacher. We had to use all of our skills that we have learned throughout the course and apply it.

    Media Project

    After being told what to create for every assignment, we were allowed to create our very own layout. I got to choose between my favorite movie, book, show, or band and create 3-4 different pages. We planned and designed the whole page with approval from our teacher and applied everything we have learned.
